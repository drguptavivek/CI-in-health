# Health as a Critical Information / Critical Infrastructure Sector
| Region              | Country                        | Status (health as CI/CII sector)                                                                                                                                                                               | Ref(s) |
| ------------------- | ------------------------------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------ |
| **Latin America**   | **Chile**                      | **Explicit (primary):** “prestación institucional de servicios de salud” listed as an **essential service** under the national cybersecurity/critical infrastructure framework                                 | [1]    |
|                     | **Mexico**                     | **Not confirmed (primary not found in accessible sources):** I did not find a government text explicitly listing “health/healthcare” as a CI/CII sector in the sources retrieved for this run                  | —      |
| **ASEAN**           | **Singapore**                  | **Explicit (primary):** CSA lists **Healthcare** among **CII sectors** under the Cybersecurity Act                                                                                                             | [2]    |
|                     | **Malaysia**                   | **Explicit (primary):** NACSA lists **Healthcare Services** as an NCII sector                                                                                                                                  | [3]    |
|                     | **Indonesia**                  | **Explicit (primary):** official summary of **Perpres 82/2022** lists **Sektor kesehatan** as a strategic sector for Vital Information Infrastructure protection                                               | [4]    |
|                     | **Thailand**                   | **Explicit (secondary):** “Public Health” presented as a CII sector under Thailand’s Cybersecurity Act (source is not a government statute page)                                                               | [5]    |
| **Africa**          | **Ghana**                      | **Explicit (primary):** CSA directive lists **Health** as a Critical Information Infrastructure sector                                                                                                         | [6]    |
|                     | **Kenya**                      | **Explicit (primary):** national coordination body list includes **health** among CII sectors                                                                                                                  | [7]    |
|                     | **Nigeria**                    | **Explicit (primary):** national cybersecurity policy/strategy text includes **Public Health and Healthcare Sector**                                                                                           | [8]    |
|                     | **Egypt**                      | **Not confirmed (primary not found in accessible sources):** a strategy launch page was found, but the **sector list including health** was not located in an accessible primary document in this run          | [9]    |
| **Middle East**     | **Bahrain**                    | **Explicit (primary/official):** National Cybersecurity Center lists **Health Services** as a CNI sector                                                                                                       | [10]   |
|                     | **Qatar**                      | **Not confirmed (primary not found in accessible sources):** I only retrieved a news report about the strategy launch (no official sector list including health found in this run)                             | [11]   |
|                     | **United Arab Emirates (UAE)** | **Explicit (official portal copy):** lists **Health services** among protected critical asset sectors                                                                                                          | [12]   |
| **Central Asia**    | **Kazakhstan**                 | **Explicit (primary):** definition/rules for “critical information and communication infrastructure” explicitly include **healthcare**                                                                         | [13]   |
|                     | **Kyrgyz Republic**            | **Explicit (primary):** definition of “critical information infrastructure” includes the field of **healthcare**                                                                                               | [14]   |
| **Southern Africa** | **Zambia**                     | **Explicit (primary):** Cyber Security Act defines “critical sector” and explicitly includes **health**                                                                                                        | [15]   |
|                     | **Botswana**                   | **Explicit (primary):** national cybersecurity strategy lists critical infrastructure sectors including **health**                                                                                             | [16]   |
| **Eastern Europe**  | **Russia**                     | **Explicit (secondary compilation of legal text):** law scope statement includes systems functioning in the sphere of **healthcare (здравоохранения)**                                                         | [17]   |
|                     | **Ukraine**                    | **Not confirmed (primary not found in accessible sources):** I retrieved official legal portals for “critical infrastructure” concepts, but not an accessible sector list explicitly naming health in this run | [18]   |
